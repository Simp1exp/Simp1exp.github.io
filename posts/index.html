    XSS # What’s this? # Cross Site Scripting or XSS is a type of injection attack. All injection attacks consist in using the input field that the application provides to enter arbitrary data. In the XSS case, the arbitrary data is normally written in JavaScript, but may also include HTML, Flash, or any other type of code that the browser may execute. There are three types of XSS: Reflected, Stored, and DOM-Based. ...
